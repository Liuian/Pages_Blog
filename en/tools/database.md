# DB 使用方法

## DBeaver
### 建表
- 找到table -> 右鍵 -> `generate SQL` -> `DDL` -> copy SOL

### 把表拿到另一台server
- 找到table -> 右鍵 -> `Export Data` -> `Database` -> `Next` -> 選到特定的ip

### 改字體大小
- `window` -> `perferences` -> `User interface` -> `appearance` -> `colors and font` -> `DBeaver Fonts` -> `Main Font`
